# 题型扩展更新 - v2.1

## ✨ 新增题型

系统现在支持 **5种题型**：

### 1. 选择题 ✅
- 单选题
- A/B/C/D 选项

### 2. 判断题 ⭐ 新增
- 对/错 二选一
- 简单直观的判断界面

### 3. 填空题 ✅
- 文本输入
- 适合概念记忆

### 4. 简答题 ✅
- 多行文本输入
- 详细解答

### 5. 编程题 ⭐ 新增
- 代码编辑器样式
- 等宽字体显示
- 深色背景，适合代码输入

## 📝 题库格式

### 判断题格式
```
判断题：计算机网络的OSI模型有7层。
答案：对

判断题：TCP协议是无连接的协议。
答案：错

# 简写格式也支持
地球是圆的。
答案：对

# 也可以在题目后加符号
判断题：数组索引从0开始。√
答案：对

判断题：栈是先进先出的数据结构。×
答案：错
```

### 编程题格式
```
编程题：编写一个函数，实现两个数的交换
答案：
def swap(a, b):
    return b, a

或使用临时变量：
def swap(a, b):
    temp = a
    a = b
    b = temp
    return a, b
```

### 选择题格式（不变）
```
1. 题目内容？
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案：A
```

### 填空题格式（不变）
```
填空题：横线处填写 ___
答案：正确答案
```

### 简答题格式（不变）
```
简答题：请简述...？
答案：详细答案
```
