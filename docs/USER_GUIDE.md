## 🚀 快速开始

### 步骤1：访问系统
```
开发环境: http://localhost:5173
生产环境: http://your-server-ip:3000
```

### 步骤2：导入题库
1. 点击顶部导航 **"导入题库"**
2. 点击上传区域选择文件
3. 支持格式：PDF、Word (.docx)、TXT
4. 等待系统自动解析

### 步骤3：开始答题
1. 导入成功后自动跳转到 **"题库列表"**
2. 选择一个题库，点击 **"开始答题"**
3. 作答后点击 **"提交答案"**
4. 答对继续，答错会加入错题本

## 🎯 功能介绍

### 1️⃣ 首页
- 显示学习统计
  - 📚 题库总量
  - ✍️ 已答题目
  - ✅ 正确率
  - 📖 错题数量
- 快速导航按钮

### 2️⃣ 导入题库
**支持的格式：**
- 📄 PDF 文件
- 📘 Word 文档 (.docx)
- 📝 文本文件 (.txt)

**题库格式要求：**
```
选择题：
1. 题目内容？
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案：A

填空题：
填空题：横线处填写 ___
答案：正确答案

简答题：
简答题：请简述...？
答案：详细答案
```

### 3️⃣ 题库列表 ⭐ 新功能
- 显示所有已导入的题库
- 每个题库卡片包含：
  - 题库名称（文件名）
  - 导入时间
  - 题目数量
  - 开始答题按钮
  - 删除按钮（右上角×）

**操作：**
- 🚀 点击"开始答题" - 只答该题库的题
- ❌ 点击"×" - 删除整个题库

### 4️⃣ 开始答题
**题型支持：**
- ✅ 选择题 - 单选
- ✅ 填空题 - 文本输入
- ✅ 简答题 - 多行文本

**功能：**
- 提交答案后即时反馈
- 答对显示 ✅ 绿色提示
- 答错显示 ❌ 红色提示 + 正确答案
- 自动记录错题
- 支持上一题/下一题导航

### 5️⃣ 错题本
**两种模式：**

**查看模式** - 复习答案
- 显示你的错误答案
- 显示正确答案
- 适合快速浏览

**练习模式** - 重新答题
- 隐藏答案
- 重新作答
- 答对后自动从错题本移除

## 💡 使用技巧

### 题库组织建议
✅ **推荐做法：**
- 按科目分开导入（便于管理）
- 文件名清晰明了
- 定期删除不需要的题库

❌ **不推荐：**
- 所有题目放一个文件（难以管理）
- 文件名模糊不清
- 题库过多造成混乱

### 答题建议
1. **首次答题** - 认真作答，测试掌握程度
2. **错题复习** - 重点关注错题本
3. **巩固练习** - 重复练习错题，直到全对

### 时间管理
- 🌅 **早上** - 新知识学习，做新题库
- 🌆 **晚上** - 复习错题本
- 📅 **考前** - 快速过一遍所有题库

## ❓ 常见问题

### Q1: 如何删除题库？
A: 在"题库列表"页面，点击题库卡片右上角的 **×** 按钮，确认即可删除。

### Q2: 删除题库会影响错题本吗？
A: 删除题库后，该题库的错题记录仍会保留，但无法再次练习这些题目。

### Q3: 可以同时导入多个文件吗？
A: 暂不支持，请一个一个导入。每导入一个文件会创建一个独立的题库。

### Q4: 题库文件有什么要求？
A: 
- 格式正确（参考上面的格式示例）
- 文件大小 < 50MB
- 编码为 UTF-8（避免乱码）

### Q5: 如何备份数据？
A: 
```bash
# 备份数据库文件
cp db.json db.json.backup

# 服务器上定时备份
0 2 * * * cp /path/to/db.json /backups/db_$(date +\%Y\%m\%d).json
```

### Q6: 如何重置所有数据？
A:
```bash
# 删除数据库
rm db.json

# 重启服务（会自动创建新数据库）
pm2 restart exam-system
```

### Q7: 多人使用会冲突吗？
A: 不会。所有用户看到相同的题库，但每个用户的答题记录和错题本是独立的。
